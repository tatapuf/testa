# Carte interactive – Tassin
Ouvrez `index.html` dans un serveur web (ex. `python -m http.server`) pour activer la géolocalisation et le service worker.

- Filtre par catégories (checkbox).
- Recherche par nom/tags.
- Cluster auto des marqueurs.
- Itinéraire piéton depuis le logement (coordonnées par défaut : 45.758247, 4.778278).

Fichiers :
- index.html
- poi_data.json
- service-worker.js (cache simple pour usage hors-ligne)
